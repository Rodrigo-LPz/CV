# CV Web · Rodrigo López

Proyecto base para construir un CV web diferencial orientado a reclutadores y empresas.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Vercel Analytics + Speed Insights

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```
